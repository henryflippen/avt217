Step 2: Create a new document with the size 750px by 1334px in portrait orientation

Step 3: create a rectangle that fills roughly two thirds of the screen and change its color to #353535 and make sure the rectangle is centered on the screen.

Step 4: create a text box that is centered on the colored box you just created and change the font size to 400 pt and the font to Source Serif Variable and change the text color to C 0% M 25.25% Y 90.22% K 0%.
